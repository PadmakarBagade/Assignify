# Assignify - Online Assignment Submission Platform

## Overview

Assignify is a full-stack web application designed to streamline academic assignment management for educational institutions. The platform enables teachers to create and manage assignments while allowing students to submit work, track deadlines, monitor performance, and compete through a leaderboard-based evaluation system.

## Features

### Teacher Module

* Create, update, and manage assignments
* Set deadlines and submission requirements
* Monitor student submissions
* Evaluate student performance
* Track assignment completion status

### Student Module

* View assigned tasks and deadlines
* Upload and submit assignments
* Track submission history
* Monitor scores and performance
* View leaderboard rankings

### System Features

* JWT-based Authentication & Authorization
* Role-Based Access Control (Teacher/Student)
* File Upload & Submission Management
* Assignment Tracking & Deadline Monitoring
* Automated Scoring System
* Leaderboard & Ranking Management
* Performance Analytics Dashboard
* RESTful API Architecture

## Installation

### Clone Repository

git clone https://github.com/your-username/Assignify.git

### Install Dependencies

Frontend:

npm install

Backend:

npm install

### Configure Environment Variables

Create a `.env` file and configure:

* Database Credentials
* JWT Secret Key
* Server Configuration

### Run Application

Backend:

npm start

Frontend:

npm start
